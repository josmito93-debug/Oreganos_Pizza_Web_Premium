# Oregano's Pizza Web Premium

Sitio web premium y funcional para Oregano's Pizza.

## Incluye
- Carrito funcional
- Upsells en carrito
- Pedido estructurado a WhatsApp
- Modal de personalización por tamaño y extras
- Delivery / take out
- Imágenes profesionales generadas
- Logo PNG generado e integrado
- Galería con artes originales del menú

## Abrir
1. Abre `index.html` en tu navegador.
2. Si prefieres una sola pieza, usa la versión autocontenida incluida en el paquete final.
